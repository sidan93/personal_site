# absidorov.ru — Personal Site

Personal portfolio of Andrei Sidorov, backend developer.

🌐 **[absidorov.ru](https://absidorov.ru)** — this site  
🤖 **[ai.absidorov.ru](https://ai.absidorov.ru)** — AI projects & notes

## Stack

- [Next.js 14](https://nextjs.org) — static export
- [Tailwind CSS](https://tailwindcss.com)
- Deployed via GitHub Pages

## Design

The site includes 10 switchable colour themes (toggle in the top-right corner):

| Theme | Light | Dark |
|-------|-------|------|
| Luxury | Warm cream + green | Pure black + purple |
| Aurora | Icy white + blue | Midnight + northern lights gradient |
| Amber | Warm beige + amber | Dark chocolate + gold |
| Terminal | Paper + green | Void black + matrix green |
| Mono | Neutral white + black | Charcoal + purple pop |
| Noir | Ivory + sienna | Void black + gold |
| Spatial | Clean white + blue | Deep space + icy blue |
| Analog | Faded paper + sepia | Dark wood + old gold |
| Swiss | Pure white + red | Pure black + red |
| Bento | Soft white + indigo | Deep charcoal + violet |
